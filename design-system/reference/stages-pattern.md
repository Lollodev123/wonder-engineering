# Three-stage narrative pattern

The public workflow uses a compact pitch structure for each rendered concept:

```text
<Hero title>
<One-sentence context>

Goals
• <verb phrase>
• <verb phrase>
• <verb phrase>

Stage ① Context
<physical movement + synchronized screen content>

Stage ② Interaction
<trigger + reveal behavior + surfaced information>

Stage ③ Focus
<one visual payoff + supporting detail>
```

## Voice rules

1. Make the installation or wall the subject.
2. Pair physical movement with screen content.
3. Describe the interaction trigger precisely when one exists.
4. End on a visible payoff rather than an abstract claim.
5. Keep each stage to one short paragraph.

This pattern is a writing aid, not copied client language. Project-specific pitch material belongs in private briefs.
